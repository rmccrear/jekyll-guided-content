# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-15

### Added
- Initial release
- Course structure with index page and lesson cards
- Lesson layout with step-by-step content organization
- Focus mode with utility bar navigation
- Table of Contents synchronization with focus state
- Custom Liquid tags: `{% level %}` and `{% showme %}`
- Pandoc converter integration
- Responsive design with Bulma CSS
- Mobile navigation support
